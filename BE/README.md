# Backend
Python version: 3.13.5  
Poetry version: 2.1.3  

### Create a virtual environment
`python3 -m venv path/to/venv`\
`source path/to/venv/bin/activate`

### Install libraries
`cd BE`\
`pip install -r requirements.txt`

### Version check
`python --version`\
`poetry --version`

### Poetry Installation
`poetry install`
